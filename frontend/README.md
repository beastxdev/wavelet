# Frontend

Wavelet's Flutter source currently lives at the repository root, which is the standard Flutter project layout.

Primary Flutter paths:

- `../lib/`
- `../test/`
- `../assets/`
- `../android/`
- `../ios/`
- `../web/`
- `../pubspec.yaml`

This directory is kept as a structure marker for teams that prefer the `frontend/`, `backend/`, and `lavalink/` mental model.
